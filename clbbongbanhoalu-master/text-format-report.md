# Vietnamese Typography & UI Formatting Report

**Date:** 2026-01-02
**Project:** CLB Bóng Bàn Hoa Lư

## 1. Applied Formatting Modes

### MODE 1 – Title Case
**Target:** Headings (H1–H6), Hero Titles, Form Labels, List Titles.
**Rule:** Capitalize the first letter of every significant word.
**Examples:**
- `Khung giờ tập luyện` → `Khung Giờ Tập Luyện`
- `Hành trình Hoa Lư` → `Hành Trình Hoa Lư`
- `Quy định tập luyện` → `Quy Định Tập Luyện`

### MODE 2 – Uppercase
**Target:** Navigation Menu Items, Primary Action Buttons, Brand Accents.
**Rule:** Convert all characters to uppercase while preserving accents.
**Examples:**
- `Trang chủ` → `TRANG CHỦ`
- `Tham gia ngay` → `THAM GIA NGAY`
- `Đăng ký hội viên` → `ĐĂNG KÝ HỘI VIÊN`
